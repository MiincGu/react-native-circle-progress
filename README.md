react-native-circle-progress
=======

Circle Progress Indicator for React Native.

## Usage

```
<CircleProgress
  progress={this.state.progress}
  lineWidth={5}
  circleRadius={20}
  circleColor='#32A6FF'
  circleUnderlayColor='#C2C2C2'
  style={{marginVertical: 20}}/>
```
